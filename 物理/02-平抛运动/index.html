笔记核心结论

  • 1)速度偏向角正切值是位移偏向角正切值的两倍：
    tan α = 2 tan β
  • 2)任意时刻，速度 V_t 的反向延长线，必定交于水平位移的中点 (x/2, 0)。
  • 1)落到斜面上时，距离斜面最远的时刻 t' 恰好是总飞行时间 t 的一半：
    t' = t / 2
  • 2)此时速度方向与斜面平行。
  • 1)化曲为直：
    水平：匀速 V_x = V_0\cos\theta
    竖直：上抛 V_y = V_0\sin\theta
  • 2)最高点：
    V_y = 0, V_x = V_0\cos\theta
    上升时间 t = \frac{V_0\sin\theta}{g}
  • 3)射高与射程：
    射高 h = \frac{V_0^2\sin^2\theta}{2g}
    射程 X = \frac{2V_0^2\sin\theta\cos\theta}{g}
