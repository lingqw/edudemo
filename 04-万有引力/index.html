笔记核心结论

G Mm / r² = m v² / r
⇒ v = √(GM / r)
G Mm / r² = m ω² r
⇒ ω = √(GM / r³)
G Mm / r² = m a
⇒ a = GM / r²
G Mm / r² = m (4π²/T²) r
⇒ T = √(4π²r³ / GM)
第一宇宙速度推导：
G Mm / R² = m v² / R
⇒ v = √(GM / R)
黄金代换：G Mm / R² = mg
⇒ GM = gR²
v = √(gR² / R) = √(gR) = 7.9 km/s

V₁=7.9 km/s 是发射卫星的最小速度，卫星运行的最大速度。

双星特征：
  • 周期 T 相同, 角速度 ω 相同
  • 距离 L = r₁ + r₂
  • 线速度 v₁ : v₂ = r₁ : r₂
  • 质量比 M₁ : M₂ = r₂ : r₁
求双星总质量 M：
G M₁M₂ / L² = M₁ (4π²/T²) r₁
⇒ M₂ = 4π²L²r₁ / GT²
同理 ⇒ M₁ = 4π²L²r₂ / GT²
总质量 M = M₁+M₂ = 4π²L³ / GT²
